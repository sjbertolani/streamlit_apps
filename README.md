# Streamlit Apps

Example Streamlit applications built on [RelationalAI](https://relational.ai) and Snowflake.

## Apps

| App | Description |
|-----|-------------|
| [`python_file_semantic_viewer/`](python_file_semantic_viewer/) | Interactive graph visualization of a RelationalAI semantic layer. Upload a PyRel model file to explore concepts, relationships, and live data from Snowflake. |
| [`observability_dashboard/`](observability_dashboard/) | RAI engine observability dashboard. |

## Requirements

Each app has its own `requirements.txt` and `environment.yml` for Snowflake deployment. See the individual app directories for setup instructions.
